# Report Generator App

The **Report Generator App** allows users to generate well-structured reports about ML Ops, download them as text files, and generate references automatically. This tool is designed to simplify report creation and improve productivity.

## Features

- **Generate Report**: Create a structured report based on user input or predefined templates.  
- **Download TXT**: Save the generated report as a `.txt` file for offline use or sharing.  
- **Generate References**: Automatically create references or citations for your report.  

## How to Use

1. **Generate Report**
   - Enter the necessary input or select a template.
   - Click the **Generate** button.
   - The app will display the generated report in the output area.

2. **Download TXT**
   - Once the report is generated, click **Download TXT**.
   - The report will be saved to your device as a `.txt` file.

3. **Generate References**
   - Click the **Generate References** button.
   - References related to your report will be displayed or added automatically.
