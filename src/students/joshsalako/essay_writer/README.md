# Agentic Report Writer

This application uses a team of AI agents to research and write a comprehensive report on any topic you provide.

## How to Use the Application

1.  **Enter a Topic**: Type the subject for your report into the "Topic" text box.
2.  **Generate the Report**: Click the **Generate Report** button to start the process. This may take a few minutes to complete.
3.  **View the Agent's Thinking**: To see the AI agents' live thought process, click on the **Click to view agent's live workstream...** accordion to expand it. You can watch as the agents plan, write, and review the report in real-time.
4.  **See the Final Report**: Once finished, the complete report will appear at the bottom of the page.
5.  **Download Your Report**: A **Download Report** button will appear below the final report, allowing you to save it as a `.txt` file.
